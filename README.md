🎨 Color Flipper

Color Flipper is a simple JavaScript project that changes the background color of the webpage when a button is clicked.

🚀 Features

* Changes background color randomly
* Displays the current color name on the screen
* Interactive button for generating new colors
* Simple and clean user interface
* Copy color

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

⚙️ How It Works

When the user clicks the **"Change Color"** button, JavaScript selects a random color from an array of colors and applies it to the background of the webpage.

This project demonstrates:

* JavaScript arrays
* Random number generation
* DOM manipulation
* Event handling

🎯 Purpose of This Project

The goal of this project is to practice JavaScript fundamentals and understand how JavaScript interacts with HTML and CSS to create dynamic web pages.
